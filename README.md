# LeetCode Solutions in C++

This repository contains my solutions to LeetCode problems, written in C++.

## Problems Solved

| Day | Problem | Approach | Complexity |
|-----|----------|----------|------------|
| 1   | Two Sum | Brute Force, Hash Map | O(n²), O(n) |
| 2   | Add Two Numbers | Linked List, Carry Handling | O(max(m,n)) |
| 3   | Longest Substring Without Repetition | Sliding Window, Hash Set/Map | O(n) |
| 4   | Longest Palindromic Substring | Expand Around Center | O(n²) |
| 5   | Median of Two Sorted Arrays | Binary Search Partition | O(log(min(m,n))) |
| 6   | Zigzag Conversion | String Traversal | O(n) |
